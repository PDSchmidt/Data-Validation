A proof of concept program that utilizes JUnit, Java, and Regular Expressions to validate data.
Edge cases are explored and tested for.

Data includes:  
Address  
City, State, Zipcode  
Currency  
Date  
Email  
Military Time  
Name (FLMI)  
Password Verification  
Phone Number (US)  
SSN  
URL  
Words that end with "ion" (Assignment specification)
